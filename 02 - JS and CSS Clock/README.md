# JS and CSS Clock

### 此題重點

1. 了解時針、分針及秒針之間，時間與角度的換算，規則如下：
    - 秒針角度 = 360 / 60 = 6 
    - 分針角度 = 360 / 60 + 秒數 * 360 / 60 / 60
    - 時針角度 = 360 / 12 + 分鐘數 * 360 / 12 / 60

其中因為分針與時針考量了偏移的狀況。

1. 秒針帶動分針的偏移角度
2. 分針帶動時針的偏移角度
